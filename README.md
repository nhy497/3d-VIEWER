# 3D 模型預覽器 🎯

一個專業的線上 3D 模型預覽工具，支援 GLB/GLTF 格式，提供完整的 3D 查看體驗。

## ✨ 功能特色

- 🎨 **現代化界面**：美觀的漸層背景和玻璃擬態設計
- 🖱️ **直觀控制**：滑鼠拖曳旋轉、滾輪縮放
- 💡 **動態光照**：可調節的光照強度，多光源系統
- 📁 **文件上傳**：支援本地上傳 GLB/GLTF 文件
- 🔲 **線框模式**：切換實體/線框顯示
- 🔄 **自動旋轉**：可調節的自動旋轉功能
- 📱 **響應式設計**：適配各種設備尺寸

## 🚀 快速開始

### 本地運行

1. 克隆或下載此項目
2. 啟動本地服務器：
   ```bash
   python -m http.server 8000
   ```
3. 在瀏覽器中訪問 `http://localhost:8000`

### 部署到 Netlify

1. 將整個項目上傳到 GitHub
2. 在 Netlify 中連接您的 GitHub 倉庫
3. 設置構建命令為空（靜態網站）
4. 發布目錄設為 `.`
5. 點擊部署！

### 部署到其他平台

此項目是純靜態網站，可以部署到任何支援靜態網站的平台：
- GitHub Pages
- Vercel
- Cloudflare Pages
- Firebase Hosting

## 🎮 使用說明

### 基本操作
- **旋轉模型**：按住滑鼠左鍵拖曳
- **縮放模型**：使用滑鼠滾輪
- **平移視角**：按住滑鼠右鍵拖曳

### 功能按鈕
- **🔄 重置相機**：恢復到預設視角
- **🔲 切換線框**：在實體和線框模式間切換
- **🔄 自動旋轉**：開啟/關閉自動旋轉
- **💡 光照強度**：調節場景光照亮度

### 文件上傳
- 點擊右上角的「📤 上傳模型」按鈕
- 選擇您的 GLB 或 GLTF 文件
- 模型將自動載入並顯示

## 🛠️ 技術棧

- **Three.js**：3D 圖形渲染
- **GLTFLoader**：3D 模型載入
- **OrbitControls**：相機控制
- **HTML5**：現代化網頁結構
- **CSS3**：響應式樣式設計
